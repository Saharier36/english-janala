# ENGLISH <img width="25px" src="./assets/logo.png" /> JANALA

An interactive English vocabulary learning platform with structured lessons, smart search, and text-to-speech pronunciation.

## 🛠 Tech Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first styling
- **DaisyUI** - Component library
- **JavaScript (ES6+)** - Interactive functionality
- **Web Speech API** - Text-to-speech
- **Font Awesome** - Icons
- **Google Fonts** - Typography

## 📡 API Used

- **Programming Hero API** - Vocabulary & lesson data

## ✨ Key Features

- 📚 Structured lessons by difficulty levels
- 🔍 Instant vocabulary search
- 🎤 Word pronunciation (text-to-speech)
- 📖 Detailed word information with synonyms
- ⚡ Loading spinner for smooth UX
- 📱 Fully responsive design
- ❓ FAQ section with toggle
